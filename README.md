# 📈 Stock API with Live Data

A simple Node.js API that fetches live stock market data using the yahoo-finance2 package.

## 🔥 Features
- Real-time stock price, EPS, P/E Ratio
- Error handling
- Easily extendable

## 🛠️ How to Run
```bash
npm install
node index.js
Visit http://localhost:3000/getStockData
